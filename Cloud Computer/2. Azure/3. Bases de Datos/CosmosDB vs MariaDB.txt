CosmosDB 
- Creado por Microsoft, release 2014
- Recomendable para BigData, tablas con solo llaves primarias
- No maneja llaves extranjeras
- Mas caro pero mas eficiente
- Para hacer Joins baje colleciones a memoria (C#) y haga join con LINQ
