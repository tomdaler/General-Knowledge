Preposesamiento

- Busca datos null

- Aplica normalizacion
- Visualiza datos

Recomendaciones
- Convoluciones 3x3
- Pooling 2x2
- Flatten (imagenes)
- Neuronas (64, 128, 256, 512) (va aumentando, segun se hace mas profundo)
- Learning rate (0.001) Adam  inicie con 0.001, pero mejor no asignarlo
- L1 L2 L1_2 (1e-5)
- Dropout (0.2)  (que % se desconecta para no memorizar)

Si a Adam no le da valor el algoritmo va automodificandose, MUCHO MEJOR!!!!
