
CATEGORIAS A ANALIZAR

LA PRIMERA CATEGORIA A EVALUAR ES POR TIPO DE DATOS
- Numericos / categoricos
- Voz
- Lenguaje
- Texto
- Imagen

PARA DATOS DE TIPOS VOZ, TEXTO, LENGUAJE, IMAGEN SE UTILIZAN ALGORITMOS ESPECIFICOS.

PARA LOS DATOS DE TIPO NUMERICOS/CATEGORICOS SE TIENEN LAS SIGUIENTES 2 CATEGORIAS
PARA EVALUAR EL MODELO DE M.L. A USAR


Categoria 1:
- Supervisados,   con target, ensenan a maquina cual es el target
- No supervisado, sin target, maquina propone


Herramientas para definir esta categoria: 
Explorar la data tiene o no columna(s) que definen grupo, target, outcome
Estadistica descriptiva, media, moda
Visualizar con diagrama de caja, boxplot
La data de iris tiene 3 valores de output, otra data solo es respuesta binaria, Yes/No


Categoria 2
- No Clasifica, no agrupa, sin grupos, tiene tendencia, los de regresion pero no logistica
- Clasifica, agrupa, data sin tendencia, no es de regresion, no estan correlacionados


Herramientas para definir esta categoria: 
Diagrama de dispersion ayuda a definir si son de clasificacion o de regresion
Diagrama SVM
Si muestra posibles grupos, o esta todo disperso, o si muestra regresion de algun tipo


PARA DATOS NUMERICOS/CATEGORICOS
ESCOGE ALGORITMO DE M.L. A ENTRENAR SEGUN COMBINACION:

- Supervisado - Clasificacion    :Arbol de decision, regresion logistica
- Supervisado - No clasificacion :Regresion lineal, polinomica, mutiple

- No Supervisado - Clasificacion    :K-means   KNN  SVM
- No Supervisado - No Clasificacion :Estadistica descriptiva, Algebra Lineal (matrices / vectores)


SI NO TIENE TARGET 
- MODELO K-MEANS LE CREA GRUPO
- LE ASIGNA EL GRUPO A CAD FILA, LE CREA NUEVA COLUMNA

kmeans = MiniBatchKMeans(n_clusters=4, batch_size=8).fit(x)
dataset['Grupo'] = kmeans.predict(x)

LUEGO QUE TIENE TARGET, SI ES CATEGORICO, USE NAIVE-BAYES, SUPERVISADO
