ALGORITMOS

Categoria 1
- Supervisados,   con target, ensenan a maquina cual es el target
- No supervisado, sin target, maquina propone

Herramienta: Explorar la data tiene o no columna(s) que definen grupo, target, outcome
             la data de iris tiene 3 valores de output, otra data solo es respuesta binaria, Yes/No

NOTA:
Self-supervised learning is similar to unsupervised learning 
because both work with data without human added labels.
The difference is that unsupervised learning uses clustering, grouping, and dimensionality reduction, 
while self-supervised learning draw its own conclusions for regression and classification tasks.


Categoria 2
- No Clasifica, no agrupa, sin grupos, tiene tendencia, los de regresion pero no logistica
- Clasifica, agrupa, data sin tendencia, no es de regresion, no estan correlacionados

Herramienta: Diagrama de dispersion ayuda a definir si son de clasificacion o de regresion
             Si muestra posibles grupos, o esta todo disperso, o si muestra regresion de algun tipo


ESCOGE ALGORITMO PARA ENTRENAR SEGUN:

Supervisado - Clasificacion     - Arbol de decision, random forest, regresion logistica, Bayes, KNN
Supervisado - No clasificacion  - Regresion lineal, polinomica, mutiple, Series de tiempo

No Supervisado - Clasificacion    - K-means, SVM
No Supervisado - No Clasificacion - Alg. Lineal


ESCENARIOS
==========

Tiene target (resultado), y target es binario (0,1) -> ARBOL DE DECISION / Redes neurales
Si valores de entrada:
- son discritos, categoricos -> ARBOL DE DECISIONES
- Son continuos, y amplio rango -> RED NEURAL


Tiene target (resultado), target con correlacion a 'X' -> REGRESION (lineal, polinomica, cuadratica)

Datos aleatorios, tiene target, son grupos/categorias -> Naive Bayes para entrenar y verificar

Es aleatorio, no tiene target -> Conozca agrupaciones, (segun figura) K-means, KNN, SVM 

Datos aleatorios, tiene agrupaciones, quiere clasificar dato a cual es su grupo MAS cercano -> KNN

Tiene 7 tipos de datos, y 5 categorias de salida -> redes neurales, primera capa con 7 nodos (ReLu), y salida con 5 nodos, (softmax)

Tiene muchas entradas, y 2 categorias de salida -> redes neurales, primera capa con no. nodos igual a numero de variables (ReLu), y salida con 2 nodos, (sigmoid / tanh)

Tiene muchas variables -> Covarianza, PCA

Clasificacion de texto  NLTK

Clasificacion de imagen -> Tensorflow, redes neurales
